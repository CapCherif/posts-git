Point de départ du projet

lors de la création du projet avec git :

git init (initialisation d'un projet git)

Etapes ci-dessous sont toujours présente a chaque modification

git add . (ajoute les modifications dans le git )

les modification doivent etre approuvé :

git commit -m "description de la tache ajouté.."

lors de la création du projet avec git :

il faut créer une branche appelé main (principale)

lier cette branche sur github (platforme)

suite:

git push (envoyer les modifications)

Les collaborateurs

git clone (copier le projet) au départ

aprés chaque modification : git pull
